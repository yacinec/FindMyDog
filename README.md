# FindMyDog

FindMyDog is an open-source, cross-platform, Javascript app.

The FindMyDog project aims to allow the dogs adoption between individuals.

## Install

```
$ git clone https://github.com/yacinec/FindMyDog.git
$ npm install
$ expo start
```
